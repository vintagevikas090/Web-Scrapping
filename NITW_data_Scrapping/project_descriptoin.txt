
# NIT Warangal Faculty Data Scraper  

## 📌 Project Overview  
This project is a web scraper designed to extract academic department and faculty details from the [NIT Warangal](https://nitw.ac.in/departments) website. Using Python and web scraping techniques, the scraper collects department-wise faculty data and stores it in a structured format for easy access and analysis.  

## 🚀 Objectives  
- Scrape the list of all academic departments at NIT Warangal.  
- Retrieve department-specific URLs for further navigation.  
- Extract faculty details for each department, including names, designations, and contact information.  
- Store the collected professor data in CSV format within a single folder.  

## 🛠️ Tech Stack  
- **Python**  
- **BeautifulSoup** (for HTML parsing)  
- **Requests** (for sending HTTP requests)  
- **Pandas** (for data handling and CSV storage)  

## 📂 Output  
The scraper gathers and organizes data such as:  
- Department names and corresponding URLs  
- Faculty names, designations, email IDs, and other available details  
- All extracted data is neatly stored in CSV format for further use
